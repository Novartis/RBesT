# Ulcerative Colitis.

Data set containing historical information for placebo arm of a phase II
proof-of-concept trial for the treatment of ulcerative colitis. The
primary outcome is remission at week 8 (binary).

## Usage

``` r
colitis
```

## Format

A data frame with 4 rows and 3 variables:

- study:

  study

- n:

  study size

- r:

  number of events

## References

Neuenschwander B, Capkun-Niggli G, Branson M, Spiegelhalter DJ.
